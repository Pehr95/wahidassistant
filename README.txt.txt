Instruktioner till att köra Your Assistant:

Öppna upp projektet i IntelliJ.
Välj Java 21 versionen.
Öppna pom.xml och högerklicka i filen därefter välj maven -> Reload project.
Därefter kör Main. 

Öppna upp webbläsaren och gå in på localhost:8080 för att komma till inloggningssidan.

Logga in med följande:
användarnamn: test 
lösenord: test
